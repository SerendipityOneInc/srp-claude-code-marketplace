**用户提问示例**:
- "查询最近7天的DAU" → **user_behavior** 或 **points_membership**
- "各试穿功能的使用量" → **tryon**
- "搜索词TOP10" → **search**
- "付费转化率" → **subscription**

### Step 2: 加载域文档

```instructions
READ metadata/domains/{识别的域}/README.md

- READ metadata/domains/points_membership/README.md
- READ metadata/domains/user_behavior/README.md
```

根据域README中的表选择指南,确定要查询的具体表。
